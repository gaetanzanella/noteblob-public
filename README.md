# NoteBlob

NoteBlob is a free and opensource markdown note editor backed by git. No proprietary formats, no complex versioning ; just a clean git client built for note taking.

## Features

- **Markdown editor** with live preview and mermaid diagram support
- **GitHub sync** — a straightforward commit, push, and merge workflow
- **Full-text search** across all your notes

## Download

[Available on the App Store](https://apps.apple.com/fr/app/noteblob/id6761648648)
